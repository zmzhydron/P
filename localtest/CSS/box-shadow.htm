box-shadow: 0 0 0 1px red, 0 0 20px 2px red;

参数 1：必需。水平阴影的位置。允许负值。

2：必需。垂直阴影的位置。允许负值。

3：可选。模糊距离。

4：可选。阴影的尺寸。

5：可选。阴影的颜色。请参阅 CSS 颜色值。

6：可选。将外部阴影 (outset) 改为内部阴影。

PS:可以添加多个boxShadow表达式：0 0 0 1px red, 0 0 20px 2px red，指代其样式的不同。

0 0 0 1px red的解释：

0:让其不产生位移、0:让其不产生位移、0：让其不产生模糊、1：让其阴影尺寸为1像素

0 0 0 1px red，总的意思就是让其产生一个和border:solid 1px red;效果相等的边框。

0 0 20px 2px red;的解释：

0:让其不产生位移、0:让其不产生位移、20px：让其产生20像素模糊、1：让其阴影尺寸为2像素

0 0 20px 2px red，意思就是让其产生一个和border:solid 2px red效果相等的边框和模糊度为20像素的阴影，并且不产生位移，定位为元素的border区域
